# Documentation de mon Portfolio - Manuel Montoya

## Présentation du projet

Ce portfolio a été conçu pour présenter mon parcours en BTS SIO (Services Informatiques aux Organisations), mes compétences techniques et mes expériences professionnelles. L'objectif est de fournir une interface claire et moderne pour les recruteurs et partenaires.

## Technologies utilisées

- **Frontend** : React 19 avec TypeScript pour un code robuste et maintenable.
- **Stylisation** : Tailwind CSS 4 pour un design responsive et personnalisé.
- **Routage** : Wouter pour une navigation fluide côté client.
- **Icônes** : Lucide React pour une interface visuelle intuitive.

## Structure des fichiers

- `src/pages/` : Contient les différentes sections du site (Accueil, Projets, Stages, etc.).
- `src/components/` : Composants réutilisables comme le Layout (navigation et footer).
- `src/contexts/` : Gestion du thème (clair/sombre).
- `src/index.css` : Configuration des variables de couleurs et des animations.

## Personnalisation du contenu

Toutes les informations (stages, projets, contacts) sont centralisées dans les fichiers `.tsx` correspondants dans le dossier `pages/`. Les données sont structurées sous forme d'objets TypeScript pour faciliter les mises à jour futures.

## Déploiement

Le projet est configuré pour être déployé sur Vercel via une synchronisation automatique avec mon dépôt GitHub.
